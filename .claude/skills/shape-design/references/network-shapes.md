# 主流网络 Shape 参考值

## 小规模（推理场景）

| 网络场景 | 常见 shape 格式 | 参考值 |
|---------|----------------|--------|
| Embedding 查询 | (B, D) | B=1~4, D=128~768 |
| 单句推理 | (B, S, D) | B=1, S=64~512, D=128~768 |
| 注意力分数 | (B, H, S, S) | B=1, H=8, S=128 |
| 小特征图 | (B, C, H, W) | B=1, C=64, H=W=7~14 |
| 一维特征 | (N,) | N=128~1024 |

## 大规模（训练/多核场景）

| 网络场景 | 常见 shape 格式 | 参考值 |
|---------|----------------|--------|
| 训练 batch | (B, S, D) | B=8~32, S=1024~4096, D=512~4096 |
| 大序列 | (B, S, D) | B=4~8, S=4096~8192, D=1024~2048 |
| 大特征图 | (B, C, H, W) | B=8~16, C=256~512, H=W=28~56 |
| 矩阵乘 | (M, N, K) | M=N=K=1024~4096 |
| 全连接 | (B, D) | B=1024~4096, D=1024~4096 |

## 维度格式映射

| 维度数 | 格式 | 典型算子 |
|--------|------|---------|
| 1D | (N,) | 一维规约/排序 |
| 2D | (B, D) | 全连接、Embedding |
| 3D | (B, S, D) | Transformer、RNN |
| 4D | (B, C, H, W) 或 (B, H, S, D) | CNN、注意力 |
| 5D | (B, C, D, H, W) | 3D CNN、视频 |
